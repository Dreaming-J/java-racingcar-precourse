### 기능 목록
- [x] 자동차 객체 생성
- [x] 0~9 사이의 값을 고르고 4이상일 경우 전진
- [x] 실행 결과 계산 및 출력
- [x] 우승자 계산 및 출력
  - [x] 여러 명일 경우 쉼표(,)로 구분

### 입력
- [x] 자동차 이름
- [x] 시도할 회수

### 예외 처리
#### 잘못된 값 입력 시 에러 발생시키고 "[ERROR]"로 시작하는 에러 메시지 출력 후 재입력 받음
- [x] 자동차 이름
  - [x] 자동차 이름은 쉼표(,) 기준
  - [x] 자동차 이름은 5자 이하
- [x] 시도할 회수
  - [x] 숫자만
  - [x] 양수만
